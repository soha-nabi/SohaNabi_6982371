Exercise 2.1: Bootstrap Directory Structure

CSS Folder

Purpose: Contains compiled CSS files

  • bootstrap.css - Unminified CSS
  • bootstrap.min.css - Minified CSS for production
  • bootstrap-grid.css - Grid system only
  • bootstrap-reboot.css - CSS reset
JS Folder

Purpose: Contains JavaScript plugins

  • bootstrap.bundle.min.js - All plugins with Popper included
  • bootstrap.min.js - Plugins without Popper
  • Individual plugin files (modal, dropdown, etc.)
Icons Folder

Purpose: Contains Bootstrap Icons SVG files

  • Over 2000+ SVG icons
  • Can be used as SVG or CSS font
  • Fully customizable (color, size)
SCSS Folder

Purpose: Source Sass files for customization

  • _variables.scss - Customizable variables
  • _mixins.scss - Sass mixins
  • Modular component files
